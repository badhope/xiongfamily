## 📋 项目简介

**熊氏宗族数字族谱系统**是一个现代化的家族文化传承平台，旨在通过数字化技术记录和传承家族历史、文化和血脉关系。系统融合了传统宗族文化与现代Web技术，为熊氏族人提供了一个便捷、美观的家族信息管理工具。

### 🌟 核心价值
- **文化传承**: 记录家族历史、字辈、家训等传统文化元素
- **血脉连接**: 通过族谱树直观展示家族血脉关系
- **互动交流**: 提供在线祭祖、许愿祈福等互动功能
- **信息管理**: 支持家族成员、事件、相册等信息的管理

## 🎯 主要功能

### 宗族文化
- **宗族起源**: 展示家族历史渊源和迁徙历程
- **家风家训**: 传承家族传统美德和行为规范
- **先贤列传**: 记录家族杰出人物事迹

### 互动功能
- **在线祭祖**: 上香祭拜，表达对祖先的敬意
- **摇签获取**: 随机获取家训，传承家族智慧
- **许愿祈福**: 写下心愿，祈求祖先庇佑

### 信息管理
- **族谱树**: 可视化展示家族血脉关系
- **谱系查询**: 快速查找家族成员信息
- **家族事件**: 记录家族重要历史事件
- **家族相册**: 保存家族珍贵影像资料
- **家族论坛**: 族亲交流互动平台

### 实用工具
- **导出族谱**: 导出家族数据
- **打印族谱**: 打印纸质族谱
- **分享功能**: 分享家族信息

## 🚀 快速开始

### 环境要求
- 现代Web浏览器（Chrome, Firefox, Safari, Edge）
- 互联网连接（用于加载外部资源）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/badhope/xiongfamily.git
   cd xiongfamily
   ```

2. **启动本地服务器**
   ```bash
   # 使用Python 3
   python -m http.server 8000
   
   # 或使用Node.js
   npx http-server -p 8000
   ```

3. **访问应用**
   打开浏览器，访问 `http://localhost:8000`

## 📖 使用指南

### 基本操作
1. **导航**: 使用顶部导航栏或页脚链接浏览不同功能模块
2. **搜索**: 在谱系查询模块输入姓名或选择辈分进行查询
3. **互动**: 点击相应按钮进行上香、摇签、许愿等操作
4. **管理**: 使用管理按钮添加家族成员、事件、相册等信息

### 功能说明
- **在线祭祖**: 点击"上香祭拜"按钮，系统会记录祭拜次数并显示动画效果
- **摇签获取**: 点击"摇签获取"按钮，系统会随机显示一条家训
- **许愿祈福**: 在文本框中输入心愿，点击"放飞心愿"按钮，心愿会显示在许愿墙上
- **谱系查询**: 输入姓名或选择辈分，点击"查询谱系"按钮查看结果
